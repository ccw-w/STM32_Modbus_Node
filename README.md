## 1. 项目简介

本项目基于 STM32F103C8T6 开发，实现了一个简易工业采集终端。系统能够采集 DHT11 温湿度数据与 ADC 模拟量，并通过 OLED 实时显示当前状态；同时通过 RS485 接口实现 Modbus RTU 通信，支持上位机读取采集数据、下发控制指令、修改报警阈值和设备地址。关键参数支持掉电保存到片内 Flash，提升了设备在现场应用中的实用性。

## 2. 主要功能

1. 采集 DHT11 温度、湿度数据
    
2. 采集 ADC 原始值并换算为电压值
    
3. OLED 显示温度、湿度、报警阈值、报警标志、通信状态、LED 状态、蜂鸣器状态和电压
    
4. 支持 Modbus RTU 03 读保持寄存器、06 写单个寄存器
    
5. 支持 LED / 蜂鸣器远程控制
    
6. 支持温湿度超限报警
    
7. 报警发生时 LED 自动闪烁，蜂鸣器自动打开
    
8. 支持通过 Modbus 修改温湿度报警阈值和从站地址
    
9. 参数修改后延时写入 Flash，实现掉电保存
    
10. 串口接收采用 `HAL_UARTEx_ReceiveToIdle_DMA()`，按帧处理 Modbus 数据
    

## 3. 硬件平台

- 主控：STM32F103C8T6
    
- 温湿度传感器：DHT11
    
- 通信接口：USART1 + RS485
    
- 显示模块：OLED
    
- 其他外设：LED、蜂鸣器、ADC 输入
    

## 4. 软件设计说明

### 4.1 数据采集

系统每 1 秒执行一次数据采集流程：

- 读取 ADC 原始值
    
- 换算电压值并写入寄存器
    
- 读取 DHT11 温湿度数据
    
- 若读取失败，则将传感器状态置为故障
    

相关数据会同步更新到 Modbus 寄存器区，供 OLED 显示和上位机读取。

### 4.2 报警逻辑

系统比较当前温度、湿度与报警上限：

- 温度超限，报警标志 bit0 置位
    
- 湿度超限，报警标志 bit1 置位
    
- 报警首次触发时，蜂鸣器自动打开
    
- 报警期间 LED 进入闪烁模式
    
- 报警解除后，退出闪烁状态
    

这样可以实现本地声光报警与远程状态监测。

### 4.3 Modbus 通信

系统实现了标准 Modbus RTU 从站的基础功能：

- `0x03`：读取保持寄存器
    
- `0x06`：写单个保持寄存器
    

通信流程包括：

- 校验从站地址
    
- 校验 CRC16
    
- 解析功能码
    
- 处理合法寄存器地址
    
- 返回应答帧
    

并对可写寄存器进行了限制，避免误写采集数据区和状态区。

### 4.4 参数掉电保存

系统将以下参数保存到 STM32 片内 Flash：

- 从站地址
    
- 温度报警上限
    
- 湿度报警上限
    

参数变化后不会立即写 Flash，而是先置保存请求标志，延时约 2 秒后再执行写入，减少频繁擦写。系统上电时会优先从 Flash 恢复参数，如数据无效则加载默认值。

## 5. 寄存器说明

### 只读寄存器

- `0x0000` 温度值，单位 0.1℃
    
- `0x0001` 湿度值，单位 0.1%RH
    
- `0x0002` ADC 原始值
    
- `0x0003` 电压值，单位 0.01V
    
- `0x0012` 传感器状态，0 正常，1 故障
    
- `0x0013` 通信状态，0 正常，1 异常
    
- `0x0030` 报警标志
    

### 可写寄存器

- `0x0010` LED 控制
    
- `0x0011` 蜂鸣器控制
    
- `0x0020` 温度报警上限
    
- `0x0021` 湿度报警上限
    
- `0x0031` 从站地址
    

## 6. 上位机测试说明

可使用 Modbus Poll 进行测试。

示例：

### 读取温湿度与电压

- 功能码：`03`
    
- 起始地址：`0x0000`
    
- 读取数量：`4`
    

### 写 LED 控制

- 功能码：`06`
    
- 寄存器地址：`0x0010`
    
- 写入值：`1`
    

### 写蜂鸣器控制

- 功能码：`06`
    
- 寄存器地址：`0x0011`
    
- 写入值：`1`
    

### 修改温度报警阈值

- 功能码：`06`
    
- 寄存器地址：`0x0020`
    
- 写入值：例如 `350`，表示 `35.0℃`
    

### 修改湿度报警阈值

- 功能码：`06`
    
- 寄存器地址：`0x0021`
    
- 写入值：例如 `750`，表示 `75.0%RH`
    

### 修改从站地址

- 功能码：`06`
    
- 寄存器地址：`0x0031`
    
- 写入值：1~247
